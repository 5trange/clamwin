Introduction

ClamWin is a graphical front-end to the ClamAV anti-virus software that runs on Microsoft Windows. An easy to use installer program is provided.

Features include:

  • Scheduler - set up scans to run at a defined time

  • Automatic virus database updates via the Internet

  • Automatic notifications of new ClamWin releases

  • Standalone Virus Scanner

  • Context menu integration to Microsoft Windows Explorer - right click on a file to scan it

  • Microsoft Outlook add-in to scan incoming and outgoing e-mails

Please note that ClamWin Antivirus does not include an on-access real-time scanner, that is, you need to manually scan a file in order to detect a virus. The Microsoft Outlook Addin, however, will delete a virus-infected attachment automatically, without any intervention from the user.
